# OpenClaw-Repo

OpenClawに関する使い方を学習・検証するためのリポジトリ

## ワークスペース設定

```bash
~/.openclaw/
├── openclaw.json      # メイン設定ファイル
├── workspace/         # エージェントのホーム
│   ├── AGENTS.md      # 行動指針
│   ├── SOUL.md        # ペルソナ・口調設定
│   ├── USER.md        # ユーザー情報
│   ├── MEMORY.md      # 長期記憶
│   ├── TOOLS.md       # ツール設定メモ
│   └── memory/        # 日次ログ
├── skills/            # カスタムスキル
└── browser/           # ブラウザデータ
```
